# luoSkills
我开发的小小Skills

## Skills

| Skill | Description |
|-------|-------------|
| **captcha-recognition** | Recognize captcha codes from images using vision LLM. Useful for automated login or form submission scenarios. |
| **feishu-custom-bot** | Send markdown card messages to Feishu custom bot via webhook. |
| **we-mp-rss-export-markdown** | Export subscribed WeChat Official Account articles from we-mp-rss platform to markdown files. |

## Usage

Each skill is independent and has its own documentation in `SKILL.md`. Visit the corresponding skill directory for detailed usage instructions.
